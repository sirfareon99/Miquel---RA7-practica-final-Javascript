1. Variables y tipos

Explicación

Las variables, como en cualquier lenguaje de programación es un elemento que contiene un valor asignado en el mismo código o preguntando al usuario let, const y var (en clasehemos visto el let y const aunque en los test solo aparecía el var).

Los tipos principales son: string (texto), number (números), boolean (true/false), float(decimales), y otros más complejos.

Ejemplo de código

let nombre = "María";
const edad = 17;
let esEstudiante = true;

console.log(nombre + " tiene " + edad + " años");

Explicación del ejemplo

Creamos tres variables de diferentes tipos y las mostramos por consola juntando el texto.

2. Operadores

Explicación

Los operadores nos permiten hacer operaciones con los datos: +, -, *, /, % (residuo), == (igual), >, <, etc. Existen operadores lógicos, de comparación, de asignación y aritmétricos

Ejemplo de código

let a = 10;
let b = 3;
console.log(a + b);     // 13
console.log(a % b);     // 1
console.log(a > b);     // true

Explicación del ejemplo

Hacemos suma, resto y una comparación.

3. Condicionales

Explicación

EL If y Else es una estructura de control condicional en que se presenta una condición para que se ejecute la parte del código cuando dicha condicición sea cumplida. Siendo la primera if, si hay más else if y ya cuando no se cumpla ninguna el else

Ejemplo de código

let nota = 8;

if (nota >= 5) {
    console.log("Aprobado");
} else {
    console.log("Suspenso");
}

Explicación del ejemplo

Si la nota es 5 o más, dice "Aprobado", si no, "Suspenso".

4. Bucles

Explicación

Los bucles repiten código varias veces. Los más usados son for que funciona con un iterador que recorre el bucle desde un inicio hasta un final e incluyendo el como puede recorrer el bucle si va 1 en 1 o 2 en 2 saltando partes y while es un tipo de bucle que mientras se cumpla la condición asignada seguirá ejecutándose.

Ejemplo de código

for (let i = 1; i <= 5; i++) {
    console.log("Número: " + i);
}

Explicación del ejemplo

Este bucle muestra los números del 1 al 5.

5. Arrays

Explicación

Un array es una lista de valores. Es un tipo de valor compuesto. Me gusta decir que es una variable con más de un valor en resumidas cuentas. Se pueden añadir, quitar o modificar elementos(los valores que tiene). Haydiferentes tipos, las listas que van en [] y se puede modificar. Las tuplas que van en () y una vez asignadas no se pueden modificar. Los sets que solo pueden tener valores únicos y va en {} y no están indexados. La indexación en una matriz se refiere a que cada elemento dentro de la misma tiene un valor oculto que indica la posición de la matriz siempre empezando por el valor 0. PD: Los Sets no tienen valores de indexación.

Ejemplo de código

let frutas = ["manzana", "plátano", "naranja"];
frutas.push("fresa");
console.log(frutas.length);

Explicación del ejemplo

Creamos un array, añadimos una fruta más y mostramos cuántas hay.

Prueba interactiva

manzana, plátano, naranja

6. Funciones

Explicación

Las funciones son bloques de código reutilizable. Se declaran con function. Existen diferentes tipos de función, una función normal y corriente sería el nombre de la función e indicar los parámetros necesarios si se requiere. Y luego funciones de flecha hereda el contexto de donde fue creada y no tienen argumentos

Ejemplo de código

function sumar(x, y) {
    return x + y;
}

console.log(sumar(5, 3)); // 8

Explicación del ejemplo

Función que recibe dos números y devuelve su suma.

9. Math

Explicación

El objeto Math tiene funciones matemáticas como redondear (floor si es redondear para abajo, ceil si es redondear para arriba) , potencia (pow), número aleatorio (random), etc.

Ejemplo de código

console.log(Math.random());        // número entre 0 y 1
console.log(Math.floor(4.7));     // 4
console.log(Math.pow(2, 3));      // 8

Explicación del ejemplo

Muestra número aleatorio, redondeo hacia abajo y potencia.

10. Strings

Explicación

Los strings son textos que van en "". Podemos unirlos, sacar partes, convertir a mayúsculas, etc.

Ejemplo de código

let texto = "Hola mundo";
console.log(texto.toUpperCase());
console.log(texto.length);

Explicación del ejemplo

Convierte el texto a mayúsculas y muestra su longitud.

13. Errores típicos

Explicación

Olvidar punto y coma, usar = en vez de ==, no declarar variables, intentar acceder a elementos antes de que cargue la página, etc.

Ejemplos comunes

  • Olvidar cerrar comillas o llaves
  • Usar == vs =
  • No usar let/const
  • Errores de mayúsculas/minúsculas
